Blender Shortcuts

G=Blöcke nehmen

R=Blöcke Drehen

S=Blöcke Skalieren

Shift+A=neues Objekt hinzufügen

7=Ansicht von oben

3=Ansicht von der seite

1=Ansicht von vorne

Shift+D=Duplizieren

Mausrad drehen=raus und rein zoomen

Mausrad gedrückt=herum bewegen

Wichtig:TAB für in den edit mode zu wechseln

Edit mode Shortcuts:

1=Ecken mode

2=Kanten mode

3=Flächen mode

Ecken mode:

Ecke auswählen dann G=Objekt verändern

Kanten mode:

wen Kante ausgewählt ist:

G=kannte verändern

Ctrl+B=Kante abflächen

Ctrl+D+Mausrad drehen=Kante abrunden

Flächen mode

Wenn fläche ausgewählt ist:

G=Fläche verändern

E=Block an fläche hinzufügen

R=Block verdrehen

I=Fläche verkleinern

Ctrl+R=Würfel mitte Skalieren

Werkzeuge:

1:E

2:i

3:ctrl+B

4:ctrl:R
